## � Quick Start

### Option 1: Try Online (Recommended)
```bash
# Visit our live platform (Coming Soon)
🌐 https://algomaster-studio-ai.vercel.app
```

### Option 2: Local Development

**Prerequisites**: Node.js 18+, Python 3.9+, Git

```bash
# 1. Clone the repository
git clone https://github.com/VenkataAnilKumar/AlgoMaster-Studio-AI.git
cd AlgoMaster-Studio-AI

# 2. Install frontend dependencies
cd frontend
npm install
npm run dev

# 3. Set up backend (new terminal)
cd ../ai-engine
pip install -r requirements.txt
uvicorn main:app --reload

# 4. Open your browser
# Frontend: http://localhost:5173
# Backend API: http://localhost:8000
```

### Option 3: Docker (Easiest)
```bash
# Run the entire stack with one command
docker-compose up -d

# Access the platform
open http://localhost:3000
```
